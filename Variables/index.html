Variables

The web-app "Variables" allows students to play around with the basic concepts of variables, datatypes, operations and literals. It creates TypeScript code from the interactions which can be copied, tested and further modified.
  1. Core Application
    Start the application without any preset and task
  2. Calculate Area
    Create a program, that uses a new variable a, to hold the size of the area of the rectangle with the width w and the heigt h. The algorithm created should work with any numerical values for w and h, do not create additional literals.
  3. Swap Values
    Swap the values of the variables x and y, without creating additional literals.
