Variables

The web-app "Variables" allows students to tinker with the basic concepts of variables, datatypes, operations and literals. It creates TypeScript code from the interactions which should be copied, tested and further modified.

Color codes:

  • red = select type or operator
  • blue = drag from
  • green = drop on
On mobile, a long press starts drag. Firefox on Android does not support drag & drop...

To start the application without any preset, click here: Core Application

Tasks

Click on the link for each task to start the app with the appropriate preset. Copy the generated code to a TypeScript project in VSCode, compile and test. Tinker with the values and the types. At the end of each task, discuss your insights in the group and record them.
  1. Operations with literal values
    Type a literal and drag it to the left side of the operation, drag another one to the right. Play around with all operators and all combination of datatypes. How can you actually produce different data types as literals?. Exception: In this task, no code gets generated.
  2. Declare and define variables
    Declare and define variables with different names and types. Drop literals on to the value fields of the variables. Explain the difference between declaration and definition.
  3. Variables and operations
    Use variables to feed the operation. Drop the result into the value field of a variable and watch the generated code.
  4. Calculate Area
    Create a program, that uses a new variable a, to hold the size of the area of the rectangle with the width w and the heigt h. The algorithm created should work with any numerical values for w and h, do not create additional literals.
  5. Oh no!
    Create a program of only three lines of code (including the line it starts with), that produces "Oh no! Oh no! Oh no! Oh no! " as the value of variable t. Do not create additional literals or variables.
  6. Swap Values
    Swap the values of the variables x and y, without creating additional literals. No operations needed
